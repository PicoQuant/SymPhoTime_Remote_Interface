Dialog

SymPhoTime 64  Server

  [Related Topics]

To open this dialog by means of the main menu click on
    "View | SymPhoTime 64  Server…".
It serves as a diagnosis tool for remote processing. For further information on the remote protocol refer to the appendix.

User Interface:

 
Element Description
IP Address SymPhoTime: Lists all IP addresses that are associated with the PC on which the SymPhoTime software is running.
Data Transfer Record Version= Displays the format version of the last received data transfer record.
Received Data Displays the data of the last measurement request:
    Group:
empty /
not transferred:		 Single measurement requested.
else: Requested measurment is is part of a group with the name shown in Group:. If this group does not yet exist, a new group is created.
    File: If transmitted: The name of the TTTR file that will store the measurement data (valid for final measurements only).
    Test Measurement
Checked: The requested measurement is a "test measurement": No data will be stored.
Unchecked:   The requested measurement is a "final measurement": Data will recorded to the workspace.
    Point Measurement
Checked: A Point Measurement, i.e. a time trace measurement, has been requested.
Unchecked:   An Image Scan has been requested.
    Image Scan
Checked: An Image Scan has been requested.
Unchecked:   A Point Measurement, i.e. a time trace measurement, has been requested.
    Bidirectional
Checked: The requested Image Scan is performed with a bidirectional scanning pattern.
Unchecked:   The requested Image Scan is performed with a monodirectional scanning pattern.
    Pixels: Displays the number of pixels in X and Y direction for an Image Scan.
    Resolution: Displays the spatial resolution for an Image Scan.
Clear Click this button to erase the remote processing messages and events log.
Events Log Logs the remote processing messages and events.
