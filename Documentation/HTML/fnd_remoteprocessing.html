Remote Processing

  [Related Topics]   

SymPhoTime is designed to work as a remote server, controlled by other applications via TCP/IP. Once started, it listens for clients that establish a connection starting with a valid server request message. This message has to contain all information as shown in the SymPhoTime Server dialog needed to accuire the data from a point or image measurement.

For detailed information on protocol and messages structure refer to the appendix.
