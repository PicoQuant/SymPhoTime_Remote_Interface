Transmission Control Protocol / Internet Protocol

Protocol suite, named after the two protocols that it containes, which were defined first and are most important until now.

Among others, TCP includes protocols for the World Wide Web (www), e-mail (SMTP), the File Transfer Protocol (FTP) etc. whereas IP is a network layer protocol dealing with data packetisation, addressing, routing etc.
